ข้อมูลนี้ถูกรวบรวมขึ้นมาเพื่อใช้ใน ระบบค้นหาแลปตรวจโควิด
โดยข้อมูลถูกรวบรวมจากหลายช่องทาง และได้ข้อมูลตั้งต้นจาก กรมวิทยาศาสตร์การแพทย์ รายชื่อคลินิกเทคนิคการแพทย์เอกชนจากสภาเทคนิคการแพทย์ รายชื่อห้องปฏิบัติการ/คลินิกเทคนิคการแพทย์ที่เปิดให้บริการตรวจเชื้อไวรัสก่อโรคโควิด 19 ด้วย Antigen Test Kit (วันที่อัพเดท 19/07/2021) ข้อมูลที่ผู้ใช้แนะนำผ่านระบบ และตาม Social Media ต่างๆ ข้อมูลอาจมีการเปลี่ยนแปลงได้ตลอดเวลา

Data Dictionary

# name description
1. t ประเภทของข้อมูล d=มาจาก data.go.th, c=มาจากการรวบรวมทางช่องทางต่างๆ
2. n ชื่อแลป
3. lat latitude
4. lng longitude
5. adr ที่อยู่
6. p จังหวัด
7. pp ประเภทของแลป pu=รัฐบาล, v=เอกชน
8. mob เบอร์โทรศัพท์
9. ema อีเมล
10. lin LINE
11. w Website
12. rm หมายเหตุ และรายละเอียดต่างๆ
13. dt รองรับการ Drive Tru
14. wi รองรับการ Walk In
15. src ลิงค์ข้อมูลอ้างอิง
16. rt รองรับการตรวจแบบ RT-PCR
17. at รองรับการตรวจแบบ Antigen test
18. ftf รองรับการตรวจแบบ Fit to fly
19. lt ประเภทการตรวจ RT=RT PCR, AT=Antigen Test, FTF=Fit to fly

20. ic สถานะการเปิดรับตรวจ ถ้าเป็น true จะไม่แสดงผลที่หน้าเว็บ
